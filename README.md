Esta es una reimplementacion de 'AttGAN: Facial Attribute Editing by Only Changing What You Want'. Para poder correr el entrenamiento es necesario descargar el dataset 'CelebAMask-HQ.zip' y descomprimirlo. Este se encuentra en 'https://drive.google.com/file/d/1badu11NqxGf6qM3PTTooQDJvQbejgbTv/view'. Los archivos correspondientes deben ponerse en la carpeta data.

En este repositorio se encuentra precargado lo necesario para atraves de 'results_.ipynb' realziar las pruebas de validaci'on. Los pesos de la red estan guardados asi como las perdidas del entrenamiento y el dataset de validaci'on en './files'
